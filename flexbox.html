CSS Flexible Box Layout, commonly known as Flexbox, is a CSS web layout model.It is in the W3C's candidate recommendation (CR) stage. The flex layout allows responsive elements within a container to be automatically arranged depending on viewport size.Most web pages are written in a combination of HTML (Hypertext Markup Language), JavaScript and CSS (Cascading Style Sheets). One of the most defining features of the flex layout is its ability to form-fit, based on its viewing environment.
